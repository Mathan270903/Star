Skills

Skills are never taught, they are acquired

HTML

HTML (HyperText Markup Language) is the standard language used to structure and format web content. It organizes text, images, and multimedia elements into a cohesive webpage layout.

CSS

CSS (Cascading Style Sheets) is used to style and layout HTML elements, controlling design aspects like colors, fonts, spacing, and positioning for visually appealing webpages.

JavaScript

JavaScript (JS) is a programming language that enables interactive and dynamic features on websites, such as form validation, animations, event handling, and real-time content updates.

React

React is a JavaScript library for building user interfaces. It enables the creation of reusable components, efficient rendering, and dynamic, interactive web applications with fast performance.

C-Sharp

C# (C-Sharp) is an object-oriented programming language developed by Microsoft. It's used for Windows applications, web development with ASP.NET, and game development using Unity.

.Net Frameworks

.NET is a software development framework created by Microsoft. It supports multiple languages, such as C#, and is used for building web, desktop, and mobile applications.

React Native

React Native is a framework for building mobile applications using JavaScript and React. It allows developers to create cross-platform apps for iOS and Android with native performance.

SQL

SQL (Structured Query Language) is a standard programming language used to manage and manipulate relational databases. It allows users to query, insert, update, and delete data efficiently.

Figma

Figma is a cloud-based design tool used for creating user interfaces and prototypes. It enables real-time collaboration, making it ideal for teams working on web and mobile app designs.
